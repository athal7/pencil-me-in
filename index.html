FAQ

Is this free?

Yes! The shortcuts are free and open source. The only cost is if you use ChatGPT beyond Apple's free tier.

Where is my data stored?

Your configuration is stored in your personal iCloud Drive. No data is sent to external servers except ChatGPT for AI processing.

Can I automate it?

Yes! Set up a personal automation in Shortcuts to run every Sunday morning and get your weekly digest automatically.

What if a source stops working?

Run Setup ‚Üí Test Sources to check which feeds are working. You can remove broken ones and find new sources with AI.

Can my spouse use it too?

Each person needs to install the shortcuts on their own device. The config is personal to each iCloud account.
